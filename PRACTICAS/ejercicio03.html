Ejercicio 3: Unidades de Medida en CSS

Este cuadro tiene un ancho fijo de 200 píxeles (px), útil para elementos con tamaño exacto.

Este bloque usa 20em, que depende del tamaño de fuente del elemento padre. Ideal para diseño fluido.

Con 20rem, el ancho se basa en el tamaño de fuente raíz. Es más predecible que `em`.

Este párrafo ocupa el 50% del contenedor, adaptándose automáticamente al tamaño del padre.

Este cuadro tiene un ancho del 30vw, es decir, el 30% del ancho de la ventana del navegador.

Este párrafo mide 20vh de alto, equivalente al 20% del alto visible del navegador.

Este bloque tiene un ancho de 5 centímetros (cm), útil para impresión o medidas físicas.

Este párrafo mide 50 milímetros (mm) de ancho. 10mm = 1cm.

Este párrafo tiene un ancho de 2 pulgadas (in). Cada pulgada equivale a 2.54 cm.

Este texto usa 18pt como tamaño de fuente, una unidad común en diseño editorial.

Este párrafo usa 2pc (picas), donde 1pc = 12pt. Se usa también en tipografía.
